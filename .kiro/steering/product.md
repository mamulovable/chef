# Product Overview

Chef is an AI-powered full-stack web application builder that specializes in backend-aware development. It generates complete web applications with built-in database, authentication, file uploads, real-time UIs, and background workflows.

## Key Features
- Full-stack app generation with integrated backend
- Built-in Convex database and real-time capabilities
- Zero-config authentication via WorkOS
- File upload and storage handling
- Background job processing
- Real-time UI updates
- WebContainer-based code execution environment

## Target Users
- Developers who want to rapidly prototype full-stack applications
- Teams needing backend-integrated solutions without complex setup
- Users familiar with React/TypeScript ecosystem

## Core Value Proposition
Chef's "magic" comes from being built on Convex's reactive database APIs, which are ideal for code generation and provide seamless full-stack integration.

## Architecture
Chef is a fork of bolt.diy with significant enhancements for backend integration and Convex-specific tooling.
